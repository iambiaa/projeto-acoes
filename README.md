# 📊 Projeto - Pipeline de Dados de Ações

Pipeline automatizado em Python para coleta, tratamento e disponibilização de dados do mercado financeiro, com link no repositorio para visualização de dados no PowerBI

## 🚀 Tecnologias utilizadas

- Python
- Pandas
- yfinance
- Fundamentus
- GitHub Actions

## 🔄 Pipeline de dados

1. Coleta de dados de mercado (yfinance)
2. Coleta de dados fundamentalistas (fundamentus)
3. Tratamento e padronização dos dados
4. Geração de arquivos Excel
5. Atualização automática via GitHub Actions
6. Consumo dos dados no Power BI

---

## 🎯 Objetivo

Demonstrar habilidades em:
- Analise de dados
- ETL (Extract, Transform, Load)
- Automação de pipelines
- Integração com ferramentas de BI
